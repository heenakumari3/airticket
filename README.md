# Air Ticket Reservation System

A simple **Air Ticket Reservation System** project that demonstrates how airline ticket booking and reservation management can be implemented using **SQL and Python**.

This project includes **database design, SQL queries, and a Jupyter Notebook implementation** for managing airline reservations.

---

## Project Overview

The **Air Ticket Reservation System** helps manage airline booking operations such as:

- Flight information management
- Passenger details
- Ticket booking
- Seat availability
- Reservation records

The system is designed to demonstrate **database modeling and query execution for a real-world airline booking scenario**.

---

## Technologies Used

- **SQL**
- **Python**
- **Jupyter Notebook**
- **Database Systems**

---

##  Features

- Create and manage airline database tables
- Store passenger and flight information
- Manage ticket reservations
- Query booking records
- Demonstrate database operations using Python
